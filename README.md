# EduVerse – Learning Management System

EduVerse is a full-stack **Learning Management System (LMS)** that allows instructors to create and manage courses while enabling students to enroll, learn, and track their progress through an intuitive platform.

The platform integrates modern technologies such as **AI-powered search, secure authentication, payment gateway integration, and role-based dashboards** to create a complete online learning experience.

# Key Features

## 🧠 AI-Powered Smart Search
- Integrated **Gemini / OpenAI API** for intelligent course search
- Helps students quickly discover relevant courses

## 🔐 Google Authentication
- Secure login using **Google OAuth 2.0**
- Simplifies user onboarding and authentication

## 💳 Razorpay Payment Gateway
- Secure online payment integration using **Razorpay**
- Allows students to purchase courses seamlessly

## ⚛️ Redux Toolkit State Management
- Efficient global state management using **Redux Toolkit**
- Ensures smooth and consistent user experience

## 📚 Student & Instructor Dashboards
- Separate dashboards for students and instructors
- Students can enroll, view lectures, and track course progress
- Instructors can create, edit, and manage courses and lectures

## 🖥️ Fully Responsive Modern UI
- Clean and modern interface
- Responsive design optimized for desktop and mobile devices

# Additional Features

### Course Management
- Create, edit, and delete courses
- Upload course thumbnails and course content

### Lecture System
- Create and manage lectures for each course
- View lecture content with structured navigation

### Reviews & Ratings
- Students can submit reviews for courses
- Helps improve course quality and user feedback

### Admin Analytics
- Dashboard includes **graphs and statistics**
- Monitor course activity and platform engagement

### Profile Management
- Update profile information

RAZORPAY_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
